Our Project: Fraud Detection System

MVP Hackathon untuk mendeteksi transaksi perbankan mencurigakan secara real-time.

Lihat Repository GitHub

System Architecture

Menggunakan FastAPI sebagai core engine yang menghubungkan React Dashboard dengan IsolationForest ML Model.

  • Real-time Transaction Scoring
  • Behavior Baseline Building

Tech Stack

BackendFastAPI & SQLite
AI/MLScikit-Learn (IsolationForest)
FrontendReact & Tailwind CSS
VisualRecharts & FontAwesome

Hybrid Fraud Engine

Kombinasi Rule-Based (30%) untuk logika bisnis dan ML Anomaly (70%) untuk pola perilaku historis.

Risk Score Formula:

final_score = min(100, ml_score + rule_score)
