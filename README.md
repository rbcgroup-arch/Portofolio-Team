# Republic Barudak Coding Portfolio

Website portfolio statis untuk team **Republic Barudak Coding**. Halaman ini menampilkan profil tim, showcase project, dan informasi singkat tentang repository project yang sedang dikerjakan.

## Preview

- Local: buka `index.html` dengan Live Server atau browser
- VPS: `http://103.197.189.114/portfolio/index.html`

## Fitur

- Hero section dengan branding tim
- Carousel member team
- Modal detail member
- Showcase project Fraud Detection System
- CTA dan footer yang sudah disesuaikan dengan identitas tim

## Struktur Project

```text
.
|-- index.html
|-- style.css
|-- script.js
|-- img/
|-- deploy/
|   `-- nginx-republic-barudak-coding.conf
`-- DEPLOY_VPS.md
```

## Menjalankan Secara Lokal

Karena ini static website, tidak perlu build step.

1. Buka folder project.
2. Jalankan dengan Live Server, atau buka `index.html` langsung di browser.

## Deploy Ke VPS

Panduan deploy tersedia di [DEPLOY_VPS.md](./DEPLOY_VPS.md).
